# Autumn Lost & Found

以真实用户名 `Isabel` 为主角的竖屏秋日失物招领决策游戏，是人物与主题同时替换后的第二个独立模板样本。

```bash
npm install
npm run dev
npm run build
```

- 游戏条目：`autumn-lost-found`
- 主角用户名：`Isabel`
- 用户 ID：暂空；已知真实用户名反查接口仍返回 `data: null`，留作后续平台联调
- 主题：雨后公园失物招领亭
- 美术：1 张角色锚点、7 张状态立绘、4 张剧情关键帧
- Remix：完整源码通过 GitHub 仓库 zip 提供，构建 `base: './'`

项目记录位于 `doc/`。平台 UUID、线上 URL、仓库和发布验证写入 `doc/publish-record.md`。
